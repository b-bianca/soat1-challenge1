## Arquivo de Variáveis de Ambiente
criar um arquivo '.env' no diretório raiz do projeto

MYSQL_USR='user'
MYSQL_PASS='user'
MYSQL_HOST='database-mysql'
MYSQL_DBNAME='restaurant'
MYSQL_CONTAINER_PORT='3306'
MYSQL_HOST_PORT='3306'
~~~
